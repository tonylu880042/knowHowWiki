# GLP-1 受體促效劑與減重

## 概述

**GLP-1（類升糖素胜肽-1）** 是人體最主要的腸泌素（Incretin），由腸道在進食後分泌。它透過促進胰島素分泌、抑制昇糖素、延緩胃排空、並在下視丘產生飽足感，同時調節血糖和體重。

由於內生性 GLP-1 會被 **DPP-4 酶立即分解**，臨床上使用的是不會被分解的 **GLP-1 受體促效劑（GLP-1 RA）**。

## 作用機制（五重效應）

```
進食 → 腸道分泌 GLP-1
         ↓
    ┌────┴────────────────────────┐
    │  1. 促進胰島素分泌          │
    │  2. 抑制昇糖素              │ → 降低血糖
    │  3. 抑制胃排空              │
    │  4. 下視丘飽足感            │ → 降低體重
    │  5. 減少食慾                │
    └─────────────────────────────┘
```

## 副作用與禁忌

### 常見副作用

噁心、腹瀉、便秘、嘔吐（通常幾天至幾周內消退）。建議從低劑量循序漸進。

### 禁忌症

- 本身或家族有**髓質甲狀腺癌**或多發性內分泌腫瘤症候群
- 對成分過敏
- 第1型糖尿病
- 重度腎功能不全 / 末期腎病
- 嚴重腸胃疾病史
